  • What do you mean by do-follow and no-follow link?

    • No-follow – No-follow is a hyperlink which instructs the search engine to not allow that hyperlink to influence the ranking of this link in the search engine’s index. In other words, it removes the ability to pass on its page rank status to any other website whose link is included in that webpage.
    • Do-follow – Do -follow is a hyperlink which instructs the search engine to allow that hyperlink to influence the ranking of this link in the search engine’s index. In other words, it does not remove the ability to pass on its page rank status to any other website whose link is included in that webpage.

  • What is the differences between bounce rate and pogo-sticking?

    • Bounce rate – When a user comes to a website, sees the first page, goes through that page then doesn’t goes to another page and leave that website. This activity is known as bounce rate.
    • Pogo sticking – When a user types a query in a search engine, clicks enter, then click on a website appeared on the result, then quickly goes back to search result page, and clicks on a different website on the result page. This type of activity usually occurs due to lack of information or immediate dissatisfaction of the user. This activity is known as Pogo sticking. This will also have a negative impact just like bounce rate. But, pogo-sticking has to do work only with search engine traffic, whereas bounce rate is bound to all traffic sources including search.

  • What is Robot.txt?

    Robot.txt is a text file whose purpose is to indicate the crawler about what should be allowed or not allowed to visit some web pages of your website. This information is not necessary to rank on search engine result pages (SERP). Just for an example, almost every website has an “about us” page, this page is not frequently updated and it is also a not much important page of your business so you might add a robot.txt syntax to this webpage so that the crawler doesn’t crawl this page.

  • Give syntax of a robot.txt file

    The syntax of the robot.txt file is –

    User agent:*(* for all crawler)

    Allow:/(File name)

    Disallow/(File name)

  • What is the formula of ROI?

    Return on investment is the ratio between the total revenue generated and cost invested while advertising a product. ROI should always be high. ROI = (Revenue – Cost) / Cost.

  • Write the formula for CTR (Click Through Rate)?

    Click through rate is the ratio of clicks by users who clicked on a particular link to the total number of users who actually viewed that webpage.

    CTR = (Total clicks / Total impressions) X 100.
